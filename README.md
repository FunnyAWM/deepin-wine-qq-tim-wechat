# Deepin-wine QQ、TIM还有微信

## 1.介绍

这是一个自动安装deepin-wine QQ或者TIM还有微信的脚本，由FunnyAWM设计，具体原理为直接抓deepin的包。

## 2.安装

1.给脚本设置可执行权限。

2.执行QQ.sh（若希望安装TIM，请执行TIM.sh）。

注意：两个脚本都带有微信安装。

3.在各启动一次QQ/TIM和微信后运行configure.sh（TIM用户运行configure-tim.sh）。

4.完成。
